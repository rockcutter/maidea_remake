## ディレクトリ詳細
### src
ソースファイルを置く
### src/core
他のモジュールが依存するファイルを置く
### src/module 
モジュールのソースを置く
### src/test
テストファイルを置く
### src/util
ユーティリティ系関数のソースを置く
### src/handler
ハンドラを置く
### src/client
client classを置く
